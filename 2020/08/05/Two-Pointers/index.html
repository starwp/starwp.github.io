Based on the above template, we take a look at a few questions of removing duplicates.

Basic Version Remove Duplicates from Sorted Array

For the basic one, we present an answer based on the template and another shorter one.

def remove(A):
    stored = 0
    for i in range(len(A)):
        if nums[i] != nums[stored]:
            stored += 1
            nums[stored] = nums[i]
    return stored + 1

The answer based on the template is shown as

def remove(A):
    stored, i = 0, 1
    while i < len(A):
        while i < len(A) and nums[i] == nums[stored]:
            i += 1
        if i < len(A):
            stored += 1
            nums[stored] = nums[i]
        i += 1
    return stored + 1

Keeping Two Version Remove Duplicates from Sorted Array II

For this one, we also display two answers. First, we show the shorter one.

def remove(A):
    if len(A) < 3: return len(A)
    stored = 2
    for i in range(2, len(A)):
        if A[i] != A[stored-1]:
            stored += 1
            A[stored] = A[i]
    return stored + 1

The next is the template one.

Removing duplicates totally

The first one is shorter version.

The second is the template one.

def remove(A):
    stored, i, cnt = 0, 1, 1
    while i < len(A):
        while i < len(A) and A[i] == A[stored]:
            i += 1
            cnt += 1
        if i < len(A):
            if cnt <= 1: # not repeated
                stored += 1
            A[stored] = A[i]
        else:
            stored -= 1
        i += 1
        cnt = 1
    return stored += 1

Removing triple, keeping double and keeping one quadruple

For this complicated problem, we only show the template version.

def remove(A):
    stored, i, cnt = 0, 1, 1
    while i < len(A):
    	while i < len(A) and A[i] == A[stored]:
            i += 1
            cnt += 1
        if cnt <= 2 or cnt >= 4:
            if cnt >= 4: j = 1 
            else: j = min(cnt,2)
            for k in range(j-1, -1, -1):
                if i-k < len(A):
                    stored += 1
                    A[stored] = A[i-k]
        elif cnt == 3:
            if i < len(A):
                A[stored] = A[i]
            else:
                stored -= 1
        i += 1
        cnt = 1
    return stored + 1

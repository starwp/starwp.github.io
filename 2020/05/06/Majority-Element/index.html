Majority Element

Posted on 2020-05-06
Can you find the majority element in linear time in place?

1. Problem statement

Given an array of size n and a number k, find all elements that appear more than n/k times.

In fact, there are several versions of this problem. For instance, the basic version is k = 2.

Assuming that the majority element appear x times， which meets that

$$\begin{equation}\label{eq1}
\frac{x}{n} > \frac{1}{k}
\end{equation}$$

then, we subtract by one on both sides, which yields that

$$\begin{equation}\label{eq2}
\frac{x - 1}{n - k} > \frac{1}{k}
\end{equation}$$

It is interesting to observe that after the number of majority element decrease by one the “dominating role” of the majority element keeps unchanged if we delete k numbers in the array simultaneously. Notice that the k numbers including the majority element should be different. In addition, as the elements appear more than n/k times, there will be at most k - 1 majority elements.

We come up with the algorithm that we pick k different numbers until the last round. The remaing in the last round are the candidates. We just check them again in a for loop, which idenfities if each appear n/k times.

A good analgy is that every round (not mean a round in for loop) we draw k different cards. To achieve that, we need counters for every drawing cards. This assures that we can find when the last round is.

So how to draw different cards with counters in each round? Taking k = 3 as an example. The code is shown in below.


def find_major_elements(nums):

    k = 3
    candidates = [0] * (k - 1)
    cnt = [0] * (k - 1)
    for e in nums:
        if candidates[0] == e:
            cnt[0] += 1
        elif candidates[1] == e:
            cnt[1] += 1        
        elif cnt[0] == 0:
            candidates[0] = e
            cnt[0] = 1    
        elif cnt[1] == 0:
            candidates[1] = e
            cnt[1] = 1
        else:
            cnt[0] -= 1
            cnt[1] -= 1

    res = []
    if cnt[0] > 0 and nums.count(candidates[0]) > len(nums) / k:
        res.append(candidates[0]) 
    if cnt[1] > 0 and nums.count(candidates[1]) > len(nums) / k:
        res.append(candidates[1]) 

    return res
